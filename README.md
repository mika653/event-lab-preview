# Event Lab — Preview

A self-contained preview of the Event Lab service offering from DFB Digital.

This page is a standalone build for stakeholder review. It will be migrated into the main [dfbdigital.com](https://www.dfbdigital.com) site after sign-off.

## What's on the page

1. **Hero & CTA** — preserved from the existing `EventLab.tsx` on dfbdigital.com.
2. **What Event Lab is** — Pre-event / Event Proper / Post-event framing.
3. **Three tiers** — Spark, Showcase, Spectacle, with full feature lists.
4. **The Team Standard** — the nine specialists every tier includes.
5. **Interactive Visualizer** — type in an event, pick a palette and a tier, watch the preview update live.
6. **Neon Harbour Festival** — a fictional Hong Kong festival showcase demonstrating the Spectacle tier.

## Stack

Single-file `index.html`. Tailwind via CDN. No build step.

## Local preview

Open `index.html` in a browser. That's it.
