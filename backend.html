A note on DFB's existing stack

Some of this you already have.

From the main dfbdigital.com codebase, several pieces overlap with what an Event Lab production stack needs:

  • · Google Analytics 4 is wired up site-wide.
  • · Vercel hosting + deployment is already in production.
  • · Email and CRM components already exist (EmailCRM.tsx) — likely a Resend or similar service is in play.
  • · A chat widget component (ChatWidget.tsx) suggests Crisp or equivalent is already integrated.

Before signing up for anything new, worth a quick audit of what's already paid for under DFB's existing tooling — this could shave $50-150 / mo off the Spectacle number above.
